# Create bootable SDCARD:
1. Download Raspbian Stretch Lite 
  a. 2017-11-29-raspbian-stretch-lite.img
2. Write to 4GB SD Card using Etcher

# Boot and update system:
1. Boot up
2. Log in as pi/raspberry
3. run `sudo apt-get update`
4. run `sudo apt-get upgrade -y`
5. run `sudo apt-get dist-upgrade -y`
6. run `sudo reboot`
7. Log in again after reboot

# Run raspi-config
1. `sudo apt-get install lightdm -y`
2. `sudo raspi-config`
- 3. Boot Options
    - B1. Desktop / CLI
        - B4. Desktop Autologin
- 4. Localization
    - I1. Change Locale
        - disable en_GB.UTF-8 
        - enable en_US.UTF-8
        - exit and choose en_US.UTF-8 as the default when prompted
- 5. Interfacing Options
    - P2. Enable SSH server
    - P3. Disable VNC server
    - P6. Disable access via serial
- 7. Advanced Options
    - A3. Change memory split to 16MB for GPU
3. clicked Finished, when prompted for reboot select no

# disable TTYs
1. `sudo vi /etc/systemd/logind.conf`
2. Uncomment "ReserveVT=6" and change it to "ReserveVT=N"

# Setup home directories and python scripts
1. run `sudo apt-get install -y git openbox xwit`
2. cd /tmp
3. run `git clone https://github.com/socallinuxexpo/scale-signs`
4. `sudo cp -R /tmp/scale-signs/client/pi-python/home/pi /home`
5. `sudo cp -R /tmp/scale-signs/client/pi-python/home/pi.save /home`
6. `sudo chown -R pi:pi /home/pi`
7. `sudo chown -R root:root /home/pi.save`
8. `sudo cp scale-signs/client/pi-python/usr/bin/* /usr/bin/`
9. `sudo chmod +x /usr/bin/resethome.sh`
10. `sudo apt-get install python-gtk2 python-webkit -y`
11. `sudo systemctl unmask rc-local`
12. `sudo vi /etc/rc.local`
13. add to the bottom of the file (above exit 0):
    /usr/bin/resethome.sh
14. `sudo reboot`
15. `sudo apt-get install xinit -y`
